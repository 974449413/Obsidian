# 1.1 配置文件的存储位置
---

git config

有三种配置文件，分别管理着系统，用户，仓库

当需要配置的某一项时，它会先在仓库中寻找，没有再去用户那里寻找，最后去系统那里寻找

存储位置：
1. /etc/gitconfig
2. ~/.gitconfig
3. .git/config

# 1.2 操作
---

参数类型：
1. 无参
	- --list：查看当前使用的所有属性
	- --list --show-origin：查看当前使用的所有属性，并标明它的来源文件
1. 属性
	- 无选项：查看当前使用的属性
	- --show-origin：查看当前使用的属性，并标明它的来源文件
2. 属性 + 值
	- 无选项：修改当前库下的属性配置
	- --system：修改系统下的属性配置
	- --global：修改用户下的属性配置
	- --local：修改当前库下的属性配置
